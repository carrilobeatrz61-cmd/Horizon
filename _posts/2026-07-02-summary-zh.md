## [首个合成细胞实现生长与分裂](https://www.quantamagazine.org/for-the-first-time-a-cell-built-from-scratch-grows-and-divides-20260701/) ⭐️ 10.0/10

由明尼苏达大学 Kate Adamala 博士领导的研究人员创造了 SpudCell，这是首个由非生命成分构建、能够生长、复制 DNA 并分裂的合成细胞，完成了完整的细胞周期。 这一突破标志着合成生物学的重要里程碑，证明类生命行为可以从完全非生命的化学成分中涌现，并在生物技术、医学和理解生命起源方面具有潜在应用。 SpudCell 拥有仅 90 千碱基对的最小基因组，并缺乏细胞骨架，而是使用一种新颖的分裂机制。该工作最初被《细胞》期刊拒绝，但已通过预印本分享，并在另一期刊审稿中。

hackernews · defrost · 7月1日 14:20 · [社区讨论](https://news.ycombinator.com/item?id=48747304)

**背景**: 合成生物学旨在从头构建人工细胞，以理解生命的基本原理。此前的研究已在合成细胞中实现了 DNA 复制和代谢，但细胞分裂一直是一个主要障碍，因为它通常需要复杂的细胞骨架。SpudCell 通过使用不同的分裂策略克服了这一难题。

## [FFmpeg 9.1 新 AAC 编码器大幅提升质量](https://hydrogenaudio.org/index.php/topic,129691.0.html) ⭐️ 8.0/10

FFmpeg 9.1 引入了一个新的 AAC 编码器，显著提升了音频质量，解决了长期存在的啁啾伪影和高比特率性能不佳等问题。 这一改进惠及广泛使用的开源工具 FFmpeg 的数百万用户，减少了对苹果 Core Audio 等专有编码器的依赖，并引发了关于编解码器标准的讨论。 新编码器目前仅支持恒定比特率 (CBR) 模式，并针对 48 kHz 采样率进行了优化，44.1 kHz 和 96 kHz 也可用但非最佳。

Cloudflare 宣布推出变现网关（Monetization Gateway），该服务允许网站运营者通过 x402 协议对任何资源（如网页、数据集、API 或 MCP 工具）收费，并以稳定币结算。 这一创新解决了网络上长期存在的无缝微交易需求，使 AI 代理和自主系统无需传统支付基础设施即可付费访问。它可能改变内容和 API 的变现方式，减少对广告模式和订阅分层的依赖。 变现网关使用 x402 开放协议，该协议基于 HTTP 402 Payment Required 状态码，并以稳定币结算。Cloudflare 已开放该服务的候补名单，其设计适用于 Cloudflare 网络背后的任何资源。
